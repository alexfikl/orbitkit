Adjacency Matrix Generation
---------------------------

This module contains some simple adjacency matrices, mostly used for testing.
If you find yourself needing more flexibility and more options,
`networkx <https://networkx.org/documentation/stable/index.html>`__ is the way to go.

.. automodule:: orbitkit.adjacency

Graph Metrics
-------------

This module contains some graph measures. The main focus is on measures for
signed weighted graphs, which are not available in most other libraries. For
other standard measures `networkx <https://networkx.org/documentation/stable/index.html>`__
is the way to go.

.. automodule:: orbitkit.metrics
